PortalSec

Secure Device Enrollment

🛡️ Why install this?

To keep your connection safe and private while using this network, your device needs to recognize our security certificate. This prevents "connection not private" errors and ensures your data stays encrypted.

📥 If the download doesn't work, copy this link, and try to download from Chrome, Safari or Google browser, but remember that you need to connect to this network, and close this page first

⬇️ Download Certificate

Windows PC

  1. Open the portalCA.pem file.
  2. Click Install Certificate...
  3. Select Local Machine and click Next.
  4. Select Place all certificates in the following store.
  5. Click Browse and select Trusted Root Certification Authorities.
  6. Click OK, Next, and Finish.

macOS

  1. Open Keychain Access (Cmd + Space).
  2. Select System (left sidebar).
  3. Drag the downloaded file into the list.
  4. Double-click the certificate > Expand Trust.
  5. Set "When using this certificate" to Always Trust.
  6. Save and enter your system password.

iPhone / iPad

  1. Download via Safari. Tap Allow on the popup.
  2. Open Settings > Tap Profile Downloaded (top).
  3. Tap Install and enter passcode.
  4. CRITICAL: Go to Settings > General > About > Certificate Trust Settings.
  5. Find the certificate and toggle the switch to ON.

Samsung Galaxy

  1. Go to Settings > Security and Privacy.
  2. Tap Other security settings.
  3. Tap Install from device storage.
  4. Select CA certificate > Install anyway.
  5. Select the portalCA.pem file and tap Done.

Xiaomi / Redmi

  1. Go to Settings > Passwords & security.
  2. Tap Privacy > Encryption & credentials.
  3. Tap Install a certificate > CA certificate.
  4. Tap Install anyway and select the file.

Infinix / Tecno

  1. Go to Settings > Security.
  2. Select Advanced > Encryption & credentials.
  3. Tap Install from storage.
  4. Choose the portalCA.pem file from your Downloads.

Huawei

  1. Go to Settings > Security.
  2. Tap More settings > Encryption and credentials.
  3. Tap Install from storage > CA certificate.
  4. Select the file and confirm.

Other Android

  1. Go to Settings > Search for "Credentials".
  2. Tap Install a certificate > CA certificate.
  3. Choose Install anyway and select the portalCA.pem file.

Linux (Terminal)

  1. Copy to store: sudo cp portalCA.pem /usr/local/share/ca-certificates/
  2. Update store: sudo update-ca-certificates
What is a CA Certificate?
It is a digital "ID card" that tells your phone or computer to trust this network. It ensures that the websites you visit are real and your data is safe from being watched.
